# Experimental Design Systems Reference

This document tears down the rules of standard web development. Forget the 8px grid. Forget 4.5:1 contrast ratios. We design for visceral impact, emotion, and motion.

---

## 1. COLOR — The Emotional Shader

We do not use safe "brand colors." We use color as a raw material, combining high-contrast hues, iridescence, or absolute darkness to command attention.

### The Avant-Garde Palette

```css
:root {
  /* Brutalist Void */
  --bg-void: #050505;
  --text-sculpture: #F2F2F2;
  --accent-acid: #CCFF00; /* Acid green, searing, hostile */

  /* Cyber-Iridescence */
  --cyber-magenta: #FF00FF;
  --cyber-cyan: #00FFFF;
  --blend-glow: mix-blend-mode: difference;
}
```

### The WebGL Integration

Whenever possible, colors should not be static hex codes. They should be generated by a fragment shader in Three.js or moving gradients in CSS:

```css
/* Generative Mesh Gradient */
.hero-canvas {
  background: radial-gradient(circle at 50% 50%, var(--cyber-magenta), var(--bg-void)),
              radial-gradient(circle at 100% 0%, var(--accent-acid), var(--cyber-cyan));
  filter: blur(100px) saturate(200%);
  animation: liquid-shift 20s infinite alternate linear;
  will-change: background-position, filter;
}

/* Use blend modes to force the typography through the noise */
.kinetic-text {
  mix-blend-mode: exclusion;
  color: var(--text-sculpture);
}
```

## 2. KINETIC TYPOGRAPHY — Massive & Fluid

Typography is not for reading paragraphs; it is for creating structure, rhythm, and tension.

### Sizing and Scale

**Never use `rem` for primary elements.** Every headline must be a massive `vw` calculation that breaks the container.

```css
:root {
  /* Extreme Typography Scale */
  --text-micro: 10px; /* Barely legible metadata */
  --text-body: 16px; /* Functional only when absolutely necessary */
  --text-h3: 4vw; /* Intrusive sub-headers */
  --text-h2: 8vw; /* Dominant section markers */
  --text-mega: 18vw; /* Screen-filling typographic sculpture */
}
```

### Font Selection — The Avant-Garde Stack

**BANNED** (Boring, systemic): Inter, Roboto, Arial, System-UI, standard Google Fonts.

**APPROVED by visual violence or extreme luxury:**
- **Brutalist / Technical**: Druk Wide, Monument Extended, Neue Machina.
- **Editorial Luxury**: Ogg, PP Editorial New, Canela.
- **Experimental / Kinetic**: Pilowlava, WT Volkolak, variable fonts heavily manipulated via CSS `font-stretch` or `font-weight` on scroll.

### Typographic Rules

```css
/* Tracking — extreme tight or extreme loose. Never default. */
.mega-title {
  letter-spacing: -0.06em;
  line-height: 0.8;
  text-transform: uppercase;
  /* Stroke only typography for layering */
  -webkit-text-stroke: 1px var(--text-sculpture);
  color: transparent;
}

.micro-data {
  letter-spacing: 0.15em;
  text-transform: uppercase;
  font-family: 'JetBrains Mono', monospace;
  opacity: 0.5;
}
```

## 3. SPATIAL RHYTHM — The Broken Grid

We do not use Bootstrap or Tailwind containers. We use absolute positioning, fixed elements, and overlapping Z-indexes to create parallax and depth.

```css
/* Break out of document flow */
.absolute-layer {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 120vw;
  height: 120vh;
  z-index: -1;
}

/* Stacking contexts as narrative tools */
.narrative-scene {
  position: sticky;
  top: 0;
  height: 100vh;
  width: 100vw;
  overflow: hidden;
  /* Use this container for GSAP ScrollTrigger pinning */
}
```

## 4. MOTION & PHYSICS — The Cinematic Engine

CSS transitions are for standard buttons. You are a director. You use **GSAP (GreenSock)** for complex choreography and **Lenis** for scroll friction.

### Mandatory GSAP Integration

```javascript
// Register core plugins for cinematic control
gsap.registerPlugin(ScrollTrigger, SplitText, CustomEase);

// Define the easing curve of the studio
CustomEase.create("avant-garde", "M0,0 C0.19,1 0.22,1 1,1");
CustomEase.create("brutal-snap", "M0,0 C0.8,0 0.2,1 1,1");

// Text Splitting & Staggered Reveal
const titleChars = new SplitText(".mega-title", { type: "chars" });
gsap.from(titleChars.chars, {
  y: 100,
  opacity: 0,
  stagger: 0.02,
  duration: 1.5,
  ease: "avant-garde",
  scrollTrigger: {
    trigger: ".narrative-scene",
    start: "top 80%",
  }
});
```

### The Three Motion Personalities

**1. MAGNETIC / FLUID**
- Heavy use of `mouseMove` tracking. Elements are dragged toward the cursor. Spring physics apply to release.

**2. BRUTAL / SNAPPY**
- Zero duration `set()` calls mixed with high-velocity `ease: "power4.out"`. Elements slam into place.

**3. CONTINUOUS / ETHEREAL**
- Endless linear rotations on 3D objects, slow WebGL distortion, noise algorithms defining the position of elements.

## 5. DESIGN CANONS — Avant-Garde Archetypes

Pick one archetype to rule the generation. Mixing them is failure.

### Canon 1: THE BRUTALIST PORTFOLIO
```
— Acid greens, deep blacks, raw monospace fonts.
— Massive scrolling marquees that move too fast.
— Images are highly distorted or noisy via WebGL.
— The cursor is a massive contrasting circle that inverts whatever it hovers over.
— Vibe: Chaotic, technical, hostile but fascinating.
```

### Canon 2: THE IMMERSIVE 3D EXPERIENCE
```
— The DOM is almost empty. The entire background is a Three.js `<canvas>`.
— Particles, fluid simulations, or a rigged 3D model that rotates on scroll.
— Minimal, elegant serif typography overlaid using CSS `mix-blend-mode`.
— Audio integration (subtle hums or spatial sound on interaction).
— Vibe: Expensive, mysterious, high-fashion or deep tech.
```

### Canon 3: THE EDITORIAL SCROLL-JACKER
```
— Horizontal scrolling triggered by vertical wheel movement.
— Huge uncropped photography moving at different parallax speeds.
— Typography is split: some fixed, some moving at double speed.
— Vibe: A high-end print magazine ripped apart and animated.
```

## QUICK REFERENCE — The "Is it boring?" Check

Before finalizing any code, grep for:
- Standard CSS `:hover` changing background color → Replace with magnetic GSAP animation or custom cursor interaction.
- A `max-width: 1200px` container → Delete it. Let elements bleed off the viewport.
- Standard scrollbars → Hide them. Implement custom Lenis scroll-jacking.
- A standard `<nav>` with `flex-direction: row` → Turn it into a massive full-screen overlay menu with staggered entry animations.
